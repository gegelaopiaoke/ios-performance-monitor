================================================================
          Android 性能监控工具 - Windows 打包指南
================================================================

✅ 好消息！项目已完全兼容 Windows 平台，可以直接打包成 exe 使用！

================================================================
  快速开始（3步完成打包）
================================================================

1. 安装依赖
   pip install -r requirements_windows.txt
   pip install pyinstaller

2. 运行打包脚本
   双击: build_windows.bat
   
   或手动执行:
   pyinstaller --onefile start_android_monitor.py

3. 获取结果
   打包好的 exe 在 dist/ 目录下

================================================================
  打包方式选择
================================================================

【推荐】Android 监控 (轻量版)
  - 文件大小: ~25MB
  - 功能: Android 设备性能监控
  - 命令: pyinstaller build_android.spec

【完整】统一监控 (iOS + Android)
  - 文件大小: ~35MB
  - 功能: 同时支持 iOS 和 Android
  - 命令: pyinstaller build_unified.spec
  - 注意: iOS 监控需要安装 iTunes

开发环境:

  ✓ pip (Python 包管理器)

================================================================
  已知问题及解决方案
================================================================

❌ RuntimeError: The Werkzeug web server is not designed...
✅ 已修复: 添加 allow_unsafe_werkzeug=True 参数

❌ 杀毒软件误报
✅ 解决: 添加到白名单或获取代码签名

❌ 模板文件未找到
✅ 解决: 使用 --add-data 参数包含资源文件
