================================================================
  📦 Windows EXE 自动构建说明
================================================================

✅ 代码已推送到 GitHub，正在自动构建两个 exe 文件！

================================================================
  下载构建好的 EXE (约 5-10 分钟后)
================================================================

1. 访问你的 GitHub 仓库
   https://github.com/你的用户名/仓库名

2. 点击顶部的 "Actions" 标签

3. 找到最新的构建任务 (应该显示绿色 ✅)
   - 如果还在构建中，会显示黄色 🟡，请等待完成

4. 点击进入构建详情

5. 滚动到底部的 "Artifacts" 区域

6. 下载两个 ZIP 文件:
   ✓ Android性能监控-Windows.zip  (~25MB)
   ✓ 跨平台性能监控-Windows.zip  (~35MB)

7. 解压 ZIP 文件得到 .exe

================================================================
  使用 EXE
================================================================

Android性能监控.exe (推荐):
  - 双击运行
  - 连接 Android 设备 (开启 USB 调试)
  - 自动打开浏览器 http://localhost:5003
  - 选择应用开始监控

跨平台性能监控.exe:
  - 支持 iOS + Android
  - iOS 需要安装 iTunes

================================================================
  系统要求
================================================================

✓ Windows 7/8/10/11 (64位)
✓ Android SDK Platform Tools (ADB)
✓ Android 设备开启 USB 调试

================================================================

构建配置文件:
- .github/workflows/build-windows.yml (GitHub Actions)
- build_android.spec (Android 版配置)
- build_unified.spec (完整版配置)
